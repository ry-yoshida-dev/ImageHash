# wavelet

## Overview

Wavelet hash object implementation for `HashMethod.WAVELET`.

## Components

| File / Directory | Role |
|------------------|------|
| [wavelet_hash.py](wavelet_hash.py) | `WaveletHash` implementation (PyWavelets), compatible with `ImageHasher` hash object usage |
| [mode.py](mode.py) | `WaveletMode` enum |
| [protocols/](protocols/) | PyWavelets typing protocols and typed module bindings |
